# Jaran Project Context

## Vision

Jaran is a real-time city awareness and civic intelligence platform for Bishkek.

The platform allows citizens to report, discover, and track city issues using a live map.

## Core Categories

### Safety

* reckless driving
* blocked roads
* dangerous areas
* aggressive behavior

### Infrastructure

* potholes
* broken lights
* flooding
* trash
* outages

### Traffic

* accidents
* closures
* congestion
* checkpoints

### Public Services

* water outages
* electricity outages
* transit disruptions

## User Types

### Guest Users

Can:

* view map
* view incidents
* browse priority issues

Cannot:

* create reports
* comment
* upload media
* confirm reports

### Authorized Users

Can:

* create reports
* upload photos/videos
* comment
* confirm incidents
* follow issues

## MVP Goals

Phase 1:

* Live map
* Incident reporting
* Image upload
* Authentication
* Incident details

Phase 2:

* Comments
* Confirmations
* Notifications

Phase 3:

* Trust system
